Instagram Spam Classifier
I built a naive Bayes classifier, a popular machine learning algorithm that applies Bayes' theorem. Coding the algorithm from scratch, it multiples each subprobability of each feature, then chooses the label with the highest probability. I trained it using a dataset from Kaggle.

Challenges I ran into were continuous features such as # of followers. Unlike discrete features, continuous ones can have an infinute number of values. To solve this, I created a density function using Gaussian distribution to calculate how many standard deviations a given feature was from the median.

Another problem I ran into were unique features with a probability of 0. Because all subprobabilities are multiplied together, multiplying by 0 could void the entire product. Therefore, I used Laplace smoothing, which "smoothes" to produce a nonzero probability.

Using cross-validation, I was able to predict whether an Instagram account was spam or not with an average of 90.33% accuracy, and a high of 90.33% accuracy. I used NumPy to shuffle tables and parse the CSV.

Project Virtus
During my internship with NSIN, I built a training playback application for Marine Corps Air Station Miramar.

Check out our video demo here.

How does it work?

After GPS and weapon orientation data has been acquired, the information is loaded into a desktop-standalone Unity application. Using the Google Maps API, the application recreates their movements in real-time. The application also has video playback functionality, multiplayer support, and slow down/speed up/pause replay features. This allows for accurate, data-driven training feedback.
